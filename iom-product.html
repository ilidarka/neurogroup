Нейромонитор ISIS
IOM System

Убедительна в своей мощности, удобстве для пользователя, гибкости и многообразии применений — система ISIS IOM многоцелевой продукт для мультимодульного интраоперационного нейрофизиологического нейромониторинга от SEP до MEP.

Посредством IONM возможен контроль двигательных и сенсорных нервов, что способствует сокращению риска повреждения нервов в ходе операции.

Нейрохирургия

Универсальное ПО
  • Програмная оболочка neuroExplorer
  • Поддержка всех интраоперационных режимов исследования: ЭМГ, ЭЭГ, ССВП, МВП, АВП, ЭВМ, ТВМ и пр.
  • Управление всеми аппаратными блоками нейромониторинга
  • Введение цифровой карты пациента и автоматическое составление протокола операции
  • Готовые сценарии исследований и встроенный реактор настроек
Универсальный нейростимулятор
  • Один нейростимулятор для всех режимов работы — кортикальной, транскраниальной или прямой нейростимуляции перифирических нервов
  • 12 универсальных каналов стимуляции током высокого напряжения — любой канал может быть настроен для своего типа и способа стимуляци
  • 1 канал для прямой нейростимуляции
  • Дозировка стимула по силе тока, все управление выполняется через оболочку NeuroExplorer
Усилители входящих сигналов
  • Каждый блок усилителя поддерживает 8 каналов записи
  • Максимальное число каналов — 128
  • Референсные блоки — для записи скальповых отведений: ЭЭГ, ССВП, АВП и ЭВП
  • Дифференциальные блоки — для гегистрации мышечной активности: ЭМГ и МВП
Расходные материалы
  • Одноразовые подкожные игольчатые и скальповые спиральные электроды — с различной цветовой кодировкой для удобства подключения
  • Биполярные и монополярные зонды, в том числе — для стимуляции коры головного мозга и черепно-мозговых нервов
  • Уникальные биполярные иглы для контроля глазодвигательных мышц, изолированные иглы для селективной записи ЭМГ с заданной мышцы
  • Кортикалные многоканальные грид-электроды Зонд Раабе для подкоркового моторного картирования
Большой экран и удобное шасси
  • Большой экран — обычный или сенсорный
  • Регулировка экрана по высоте и углу поворота
  • Мобильная стойка с лотками для аксессуаров
  • Мощный компьютер с принтером и аудиосистемой
Специальные аксессуары
  • Эргономичные коммутационные блоки для стимуляции и регистрирующих электродов
  • Устройство звуковой и световой стимуляции
  • Отличная экранировка всех переферийных блоков, кабелей и адаптеров
  • Мощный компьютер с принтером и аудиосистемой

Хирургия позвоночника

Регистрация Д-волны
  • Уникальный гибкий эфидуральный биполярный электрод для регистрации сигнала с поверхности спинного мозга
  • Методика Д-водны — модификация МВП, выполняемая при невозможности регистрации ответов с мышц ног — например, при использовании миорелаксантов
Малоинвазивные методики
  • Адаптер, позволяющий проверять безопасность установки перкутанных транспедикулярных систем
  • Биполярная схема стимуляции
  • Только многоразовый вариант исполнения
Стимуляционные зонды
  • Монополярный зонд для стимуляции канала транспедикулярноговинта или его головки после установки
  • Биполярны суперселективный концентрический зонд для высокоточной стимуляции спинальных корешков
Автоматизированный сценарий
  • Специальный сценарий для контроля установки систем транспендикулярной фиксации
  • Автоматическое изменение силы тока во время исследования и визуальное представление результата по принципу «светофора»
  • Выбор преднастроек в зависимости от уровня установки имплантантов
  • Мощный компьютер с принтером и аудиосистемой
Мониторинг при коррекции сколиоза
  • Нейромониторинг при хирургической коррекции сколиоза — наиболее сложный в хирургии позвоночника
  • Регистрация ССВП и МВП — обязательна
  • Оперативный контроль состояния корешков осуществляется при помощи ЭМГ в режиме «без стимуляции»
Возможность выбора
  • Аппарат на облегченной стойке ISIS IOM «Compact» — 24 канала или мобильный вариант «Еxpress» — 16 каналов
